## ✨ 주요 기능

- **🚪 방 코드 입장** — 6자리 코드만으로 누구나 즉시 참여, 설치 불필요
- **💬 실시간 채팅** — 게임 중 채팅 및 모바일 채팅 미리보기
- **👑 방장 시스템** — 방장 위임, 오프라인 시 자동 위임 및 타이머 이어받기
- **👁️ 관전 모드** — 게임 중 관전 후 다음 판 참여 신청 가능
- **🏆 누적 점수** — 게임 간 총점 누적 및 표시
- **🎉 승자 모달** — 게임 종료 시 전면 팝업으로 즉시 승자 확인
- **📱 모바일 최적화** — 반응형 레이아웃, 진동 피드백, 터치 최적화
- **⏱️ 게임별 타이머** — 오목·타이핑·빙고 30초, 그 외 10초 독립 운영

---

## 🚀 사용 방법

```
1. 사이트 접속 → 이름 입력
2. [새 방 만들기] → 방 코드 공유
3. 친구는 방 코드 입력 후 [참여하기]
4. 방장이 게임 선택 → [게임 방 생성]
5. 참가/관전 선택 → 게임 시작!
```
